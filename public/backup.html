Namespace Backup
Backup Format
This will backup the json configuration files for each of the object types listed below within the selected tenant and namespace. Optionally the 'shared' namespace can also be included in the backup.

Backed up object types:
http_loadbalancers, tcp_loadbalancers, app_firewalls, origin_pools, healthchecks, ip_prefix_sets, bgp_asn_sets, service_policys, rate_limiter_policys, routes

All files will be combined into a Zip file.
  • Zip file name: tenant_namespace_[shared]_utc.zip
  • All configuration files will be named with the format: tenant_namespace_objecttype_itemname.json
  • A manifest.json file will be created in the same directory as the backup files, detailing the included files
  • The format of this file is: { "object_type": [ { "tenant": "tenant", "namespace": "namespace", "name": "name", "filename": "filename", "tenant_namespace_type_name.json" "resource_version": "resource_version", "uid": "uid", "modification_timestamp": "modification_timestamp" }, ... ] }
